# Retail Performance Overview Dashboard
An interactive Power BI dashboard analyzing retail sales, supply chain, 
and customer performance, built with a custom UI/UX layout designed in Figma.

##  Project Overview
This project analyzes retail performance across sales, supply chain 
efficiency, and customer segments. It tracks key metrics like revenue, 
profit margin, delivery performance, and discount impact to help 
stakeholders identify trends and operational bottlenecks.

## Tools & Technologies
- **Data Source:** Kaggle (retail dataset)
- **Data Cleaning:** Microsoft Excel, Power Query
- **UI/UX Design:** Figma (custom dashboard layout design)
- **Visualization:** Power BI
- **DAX:** Calculated measures for KPIs (profit margin, late delivery rate, etc.)

##  Project Workflow
1. **Data Sourcing** – Downloaded raw retail dataset from Kaggle
2. **Initial Cleaning** – Cleaned and structured data in Excel
3. **ETL with Power Query** – Imported into Power BI, transformed and 
   shaped data using Power Query (handling nulls, data types, merging tables)
4. **UI/UX Design** – Designed the dashboard layout, color theme, and 
   component placement in Figma before building it
5. **Dashboard Build** – Recreated the Figma design in Power BI using 
   custom visuals, themes, and layout
6. **DAX Measures** – Created calculated metrics for profit, margin %, 
   delivery rate, and shipping delay
7. **Interactivity** – Added slicers, tooltips, and drill-through for 
   region/category/segment analysis

## Dashboard Key Areas
- **Sales** – Revenue trends, category performance, regional sales
- **Supply Chain** – Delivery performance, shipping delays, late 
  delivery by region
- **Customers** – Segment-wise performance and discount-profit analysis

##  Key Insights
- Central Africa has the highest late delivery rate (~58%)
- The Consumer segment generates the most revenue, making it the priority group for retention and marketing.
- Discount rate vs profit shows Higher discounts don't clearly translate into higher profit, suggesting some categories are  overdiscounted.
